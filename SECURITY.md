# Security Policy

## Supported Versions

The latest released version of Shared Albums for Google Photos (by JanZeman) on WordPress.org and this repository is supported with security updates.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly by emailing:

- <jan.zeman@gmail.com>

Please include as much detail as possible (WordPress version, PHP version, reproduction steps, and any proof-of-concept).

We will acknowledge your report, investigate, and aim to provide a fix as soon as reasonably possible.
